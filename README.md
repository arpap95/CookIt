# CookIt Recipe Application

## Overview

CookIt is a web application that allows users to add, view, update, and manage their recipes. Users can search for recipes, mark recipes as favorites, and generate reports on their recipe collection.

## Features

1. **Add New Recipes**: Users can add new recipes by providing a name, description, and category.
2. **View Recipes**: Recipes are displayed in a list, showing their name, description, and category.
3. **Update Recipes**: Users can update the description of a recipe.
4. **Favorite Recipes**: Users can mark recipes as favorites and toggle this status.
5. **Delete Recipes**: Recipes can be deleted from the list.
6. **Search Recipes**: Users can search for recipes by name or description.
7. **Sort Recipes**: Recipes can be sorted by date and category.
8. **Filter Favorite Recipes**: Users can view only their favorite recipes.
9. **Generate Report**: A report can be generated showing the total number of recipes, number of favorite recipes, and number of non-favorite recipes.
10. **Persist Data**: Recipes are stored in the browser's local storage, ensuring data persists across page reloads.

### Usage

1. **Adding a New Recipe**:
   - Fill in the recipe name, description, and select a category.
   - Click the "Add New Recipe" button.
   - The new recipe will appear in the list.

2. **Viewing and Managing Recipes**:
   - Recipes are displayed in a list. Each recipe shows its name, description, and category.
   - To update a recipe's description, click the edit icon next to the recipe, enter the new description, and confirm.
   - To delete a recipe, click the trash icon next to the recipe.
   - To mark a recipe as a favorite, click the star icon next to the recipe. Click again to unmark.

3. **Searching for Recipes**:
   - Enter a search term in the search input field.
   - Click the "Search" button or press Enter.
   - Recipes matching the search term will be displayed.

4. **Sorting and Filtering Recipes**:
   - Click the sort/filter dropdown button (down arrow).
   - Select "Show All" to view all recipes sorted by date.
   - Select "Sort By Favorite" to view only favorite recipes.
   - Select "Sort By Category" to view recipes sorted by category.

5. **Generating a Recipe Report**:
   - Click the "Generate Recipe Report" button.
   - The report will display the total number of recipes, the number of favorite recipes, and the number of non-favorite recipes.
